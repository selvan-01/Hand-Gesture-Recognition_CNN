# Hand Gesture Recognition using CNN and OpenCV

A Deep Learning project that detects and classifies hand gestures using a Convolutional Neural Network (CNN).  
The system can recognize different hand gestures such as numbers using image classification.

---

## Project Overview

Hand Gesture Recognition is widely used in:

- Human Computer Interaction
- Sign Language Recognition
- Virtual Reality
- Gaming Control Systems
- Smart Interfaces

This project uses **Deep Learning (CNN)** to classify hand gestures from images and **OpenCV** for real-time detection.

---

## Gesture Classes

The model recognizes the following gestures:

- NONE
- ONE
- TWO
- THREE
- FOUR
- FIVE
